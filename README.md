# Minimalist Pomodoro Timer

究極にミニマリストなポモドーロタイマーです。
文字や数字を一切排除し、色とアニメーション、そして触覚フィードバックだけで時間を管理します。

## 特徴

- **完全なミニマリズム**: 画面には円が一つだけ。
- **直感的なインターフェース**:
    - タップ: スタート / 一時停止
    - 長押し: リセット
- **視覚的なフィードバック**:
    - **集中モード (25分)**: ダークグレー (#696969)
    - **休憩モード (5分)**: ライトグレー (#F5F5F5)
    - **呼吸アニメーション**: タイマー動作中は円がゆっくりと呼吸するように動きます。
    - **通知音**: モード切り替え時にシンプルな電子音が鳴ります。
- **PWA対応**: スマートフォンにインストールして、オフラインでもアプリとして利用可能です。

## 使い方（スマートフォンでの実行方法）

このアプリはPWA (Progressive Web App) として設計されており、ホーム画面に追加することでネイティブアプリのように動作します。

**Web版URL:** [https://torusk.github.io/zen-pomodoro/](https://torusk.github.io/zen-pomodoro/)
（※GitHub Pagesの設定が必要です）

### iPhone (iOS) の場合
1. Safariでこのページを開きます。
2. 画面下部の「共有」アイコン（四角から矢印が出ているアイコン）をタップします。
3. メニューをスクロールして「ホーム画面に追加」を選択します。
4. 右上の「追加」をタップします。
5. ホーム画面に追加されたアイコンからアプリを起動します。

### Android (Chrome) の場合
1. Chromeでこのページを開きます。
2. 画面右上のメニューアイコン（3点リーダー）をタップします。
3. 「アプリをインストール」または「ホーム画面に追加」を選択します。
4. 指示に従ってインストールします。
5. ホーム画面に追加されたアイコンからアプリを起動します。

## 開発

### ローカルでの実行

```bash
# サーバーを起動（例: Pythonを使用する場合）
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセスしてください。

### ファイル構成

- `index.html`: アプリの構造
- `style.css`: デザインとアニメーション
- `app.js`: タイマーのロジックとPWA制御
- `sw.js`: オフライン対応用のService Worker
- `manifest.json`: PWA設定ファイル
